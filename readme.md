# Expense Tracker

A simple expense tracker built with JavaScript while learning Git and GitHub.

`Expense Tracker` is a simple JavaScript project to practice Git commands and version control. It allows users to track their expenses and manage their budget effectively.

## Status
Currently learning Git and GitHub 

## Features

- Add expenses with name and amount
- Calculate toatl expense
- Budget Limit checker with warnings

## How to Run

1. Clone this repository
2. Open expense-tracker.js in Vs Code
3. Run in terminal:

```javascript 
node expense-tracker.js
```

## Commands Used in This Project

| Command | Description |
|---|---|
| `git init` | Initialize repository |
| `git add` | Stage changes |
| `git commit` | Save permanently |
| `git branch` | Create branches |
| `git merge` | Merge branches |
| `git push` | Push to remote |
| `git pull` | Pull from remote |
| `git status` | Check status |
| `git log` | View commit history |
